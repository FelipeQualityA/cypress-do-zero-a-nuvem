# 🌲 Cypress: Do Zero à Nuvem ☁️

Este repositório contém o projeto prático desenvolvido para o curso **"Cypress, do Zero to à Nuvem"** da escola online *Talking About Testing*. O objetivo é demonstrar a automação de testes fim-a-fim (E2E) de forma profissional, cobrindo desde a base até a execução em nuvem.

---

### 🛠️ Pré-requisitos

Para clonar e executar este projeto, você precisará ter instalado:

* **Git** (v2.42.1 ou superior)
* **Node.js** (v20.13.1 LTS ou superior)
* **npm** (v10.8.1 ou superior)


---

### 🚀 Instalação
  Instale as dependências de desenvolvimento executando o comando abaixo no seu terminal:
```npm install```

## 🧪 Executando os Testes

Este projeto permite a execução em Desktop e Mobile.

### 🖥️ Desktop
- **Terminal (Headless):** `npm test`
- **Interface (GUI):** `npm run cy:open`

### 📱 Mobile
- **Terminal (Headless):** `npm run test:mobile`
- **Interface (GUI):** `npm run cy:open:mobile`
